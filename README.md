# clone-tabnews

clone do projeto tabnews.com.br (curso.dev)

> ## Dia 3

- to define the project default nvm version
  - create the file .nvmrc
- to define node version
  - <code>nvm alias default lts/hydrogen</code>
- install (using the alias default)
  - <code>nvm install</code>
- install the packages
  - <code>npm install #package (npm install next)</code>

> ## Dia 4

- change the package.json adding in the scripts part the alias dev with
  - <code>next dev</code>
- to run the project, in the terminal run
  - <code>npm run dev</code>
- created the folder pages and the file index.js with a function and declared the function in the file using export #function-name#
- with a folder and js file, the command <code> npm run dev </code> works.
- it's possible to change de visibilty of the localhost created in ports tab.
